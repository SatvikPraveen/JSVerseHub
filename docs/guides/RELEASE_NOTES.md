## ðŸŽ¯ Getting Started

### Installation

```bash
# Clone repository
git clone https://github.com/jsversehub/jsversehub.git
cd jsversehub

# Install dependencies
npm install

# Verify installation
npm run validate
```

### Development

```bash
# Start development server
npm run dev

# Run tests
npm run test:watch

# Check code quality
npm run lint

# Build for production
npm run build
```

### Deployment

```bash
# Docker deployment
docker-compose up -d

# Traditional server
npm install --production
npm start

# PM2 process manager
pm2 start ecosystem.config.js
```
